== README

# System Requirements:
* Bundler 1.10.6
* Ruby 2.2.0
* Rails 4.2.4
* MongoDB 2.6.11

# How to start server:
* bundle install
* rails server
* Access APIs from: http://localhost:3000/birds

# How to run the test suite:
* rspec .

# Postman Collection for hitting APIs:
